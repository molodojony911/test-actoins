# Test Backend

Небольшой сервис на **FastAPI**: время сервера и проверка работоспособности.

## Запуск

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

Сервис по умолчанию: http://127.0.0.1:8000

## Эндпоинты

| Метод | Путь     | Описание                          |
|-------|----------|-----------------------------------|
| GET   | `/time`  | Текущее локальное время (ISO 8601) |
| GET   | `/health`| Статус `ok`                       |

Документация API: http://127.0.0.1:8000/docs
